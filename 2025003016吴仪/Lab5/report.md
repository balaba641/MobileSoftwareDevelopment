## 三、界面结构与核心代码
### 1. 整体布局结构
本次实验采用 垂直 + 水平嵌套布局，结构清晰、层次分明：
- 根布局：Column 铺满屏幕，设置水平内边距，整体居中。
- 上半部分（展示区）：Column 权重占满，内部垂直居中，包含：
  - Image 图片展示组件
  - 淡蓝色背景文字卡片（标题 + 作者）
- 下半部分（控制区）：Row 水平排列两个按钮。

### 2. 关键技术点
- 状态管理：使用 remember { mutableStateOf(1) } 保存当前图片序号。
- 动态加载：使用 when 表达式根据序号切换图片、标题、作者。
- 界面美化：使用 background 设置淡蓝色背景，padding 设置内边距。
- 循环逻辑：按钮点击判断边界，实现 1↔2↔3 无限循环切换。
- 组件对齐：使用 Alignment 和 Arrangement 实现居中布局。

## 五、遇到的问题及解决方法
### 问题 1：添加背景色时报错，提示未找到方法
- 原因：没有导入 background 和 Color 相关依赖包。
- 解决：添加对应的导入语句，报错消失。

### 问题 2：图片显示拉伸、比例失调
- 原因：未设置宽高比，图片强制填充容器导致变形。
- 解决：给 Image 添加 aspectRatio(0.8f) 固定宽高比，保持原图比例。

### 问题 3：文字区域没有背景卡片效果
- 原因：直接显示 Text，未用容器包裹并设置背景。
- 解决：用 Column 包裹标题和作者，添加淡蓝色背景实现卡片效果。

### 问题 4：界面不居中，元素靠左或拥挤
- 原因：未设置对齐方式与内边距。
- 解决：添加对齐设置与内边距修饰符。
